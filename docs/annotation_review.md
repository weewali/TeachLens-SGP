# Annotation Review

Use this file to review the first labeled sample images before full dataset labeling continues.

## Review Checklist

- Instructors are labeled correctly.
- Students are labeled correctly.
- Bounding boxes are tight enough.
- No students are mislabeled as instructors.
- No instructors are mislabeled as students.
- Blurry or unclear frames are removed or marked.
- Every image has a matching `.txt` label file.
